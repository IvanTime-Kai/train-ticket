  • id · CHAR(36) PK
  • name · VARCHAR

// booking flow — end to end
1. Search Trip
  • 01GET /trips/search?from=HAN&to=SGN&date=...
  • 02train-service query DB
  • 03Trả về danh sách chuyến + ghế trống
2. Hold Seat
  • 01POST /bookings/hold
  • 02booking-service gRPC → train-service
  • 03Redis SET NX seat_hold:tripID:seatID TTL 5m
3. Create Booking
  • 01POST /bookings
  • 02booking-service tạo booking (pending)
  • 03Kafka publish: booking.created
4. Payment
  • 01payment-service consume: booking.created
  • 02Xử lý payment
  • 03Kafka publish: payment.success
  • 04booking-service update: confirmed
